# EmotionVR
ロボットが人の感情を認識するために，脳波の機械学習を行いたい．
しかし，現在，人の感情を喚起するために用いられているのは写真である．
そのため，より没入できる，かつ写真よりも自由度が高く作成可能であるという点で，
感情を喚起するためのVRを作成する．  
Fear:アイテムを集めながら病院からの脱出を目指す．
